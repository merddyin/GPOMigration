# GPOMigration Acknowledgements

Project Site: [https://github.com/merddyin/GPOMigration](https://github.com/merddyin/GPOMigration)

This project owes some acknowledgements to other projects. Here are some other authors or projects which have made this project possible. If you believe you or a project should be included in this list please let us know.

[ModuleBuild](https://github.com/zloeber/ModuleBuild) - A PowerShell module build framework.

[SDM Software — SDM-GPMC](https://sdmsoftware.com) - Free PowerShell module for reading Group Policy Object settings. Used as a bundled plugin to enumerate Administrative Template and Preference settings from live GPOs.

[OfficeIMO.Excel](https://github.com/EvotecIT/OfficeIMO) by Evotec IT - MIT-licensed .NET library for creating and reading Excel (.xlsx) files without requiring Microsoft Office or COM. Used to produce formatted migration workbooks with per-GPO table sheets and a TOC.
